## 6. Architecture Compliance

본 프로젝트는 Rust CLI 애플리케이션으로, Starter 레벨 구조를 따른다.

| 설계 레이어 | 설계 모듈 | 구현 모듈 | 상태 |
|------------|----------|----------|------|
| CLI (진입점) | `cli.rs`, `main.rs` | `cli.rs`, `main.rs` | ✅ |
| 분석 엔진 | `git/` | `git/` | ✅ |
| 점수 계산 | `scoring/` | `scoring/` | ✅ |
| 출력 | `output/` (terminal, json, svg, badge) | `output/` (terminal, json, svg, badge) | ✅ |
